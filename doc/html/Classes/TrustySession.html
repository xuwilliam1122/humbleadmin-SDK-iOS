Overview

通过 TrustySession，你的 App 同服务器进行通信，并完成绝大多数的用户操作，如：注册、登录、鉴权。

TrustySession 不仅保存了用户系统功能的配置信息， 还保存了注册、登录、鉴权成功后的安全令牌、存储终端用户信息的对象。
